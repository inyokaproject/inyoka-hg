Inyoka

Was treibt ubuntuusers an?

{% endblock %} {% block portal_content %}

Seit März 2008 läuft das gesamte ubuntuusers Portal auf einer einzigen Python-Anwendung mit dem Namen „inyoka“ (zulu für „Schlange“). Der Quellcode wird vorraussichtlich nach dem django 1.0 Release als OpenSource Software unter der GPL veröffentlicht.

Inyoka selbst baut dabei auf eine Vielzahl von OpenSource-Programmen, -Bibliotheken und -Technologien auf:
